Data source

Data for the plot is taken from the official webpage of the Robert Koch Institute (https://www.rki.de/DE/Content/InfAZ/N/Neuartiges_Coronavirus/Fallzahlen.html). This page is checked every hour for new data. Initial data gathering was done on 2020-03-07 using the Wayback Machine (https://web.archive.org).
